# Paint

## Environment

This software is developed, compiled and tested on the following platform:

- OS: Ubuntu 18.04.2 LTS
- C++ Compiler: g++ 7.3.0
- make: GNU Make 4.1

## Build

To build the software, simply type

```bash
make
```

The target executable is  `build/paint`.
